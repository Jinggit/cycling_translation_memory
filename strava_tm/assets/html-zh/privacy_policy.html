Strava 尊重您的隐私权，与您同样关注您可能通过 www.strava.com 目前提供访问的 Strava 网站和相关服务（统称“网站”）提交给 Strava 的任何数据的安全性。以下声明（简称“隐私政策”）揭示了 Strava 为保护您在访问网站期间向 Strava 提供的信息而制定的指导原则。

Strava 会收集我的哪些信息？它会如何使用这些信息？

仅在您自愿选择与 Strava 共享您的个人信息的情况下，Strava 才会收集此类信息。Strava 不会以本隐私声明中未提及的方式将涉及 Strava 会员或 Strava 网站其他用户的信息共享、租赁或销售给第三方。

要使用网站的核心功能，您必须首先在网站上填写注册表，并同意我们的“条款和条件”。在注册期间，您需要提供个人信息，例如您的姓名和电子邮件地址。Strava 不会刻意收集 13 岁以下儿童的信息。如果我们或第三方服务提供商要求您提供出生日期来订购特定的产品或服务，未满 13 岁者将无法输入出生日期。

本网站还有其他需要您与 Strava 分享个人信息的部分，其中可能包括您的设备和活动、骑行路线、姓名、邮编及电子邮件。如果您选择成为 Strava 会员，Strava 还会收集您的信用卡或其他付款信息。如果您选择与 Strava 共享任何个人信息，Strava 可能会存储该信息，并将其用于内部研究和其他 Strava 市场营销目的，包括向您发送电子邮件告知特别的促销活动和您感兴趣的其他计划。如果是在网站评估过程中发送的个人信息，Strava 可能还会使用该信息来联系您，以听取您的意见。如果您在注册期间选择接收 Strava 新闻通讯，Strava 将使用您的电子邮件地址向您发送新闻通讯，让您了解 Strava 的最新动态和消息。您可以通过“帐户”区域更新您的个人资料或向 support@strava.com 发送电子邮件，从我们的电子邮件列表中移除您的名字。Strava 不会以本隐私声明中未提及的方式将您的信息共享、租赁或销售给其他公司。

在您访问网站时，会有 cookie 存储在您的计算机上。通常，cookie 是通过为您的计算机指定一个在 Strava 之外没有意义的唯一编号来发挥作用。Strava 的 cookie 不包含任何可以识别个人身份的信息；Strava 使用它们来提供个性化之类的功能，保存您的会员 ID 及跟踪您的购物车。Strava 可能还会使用 cookie 来协助提供您特别感兴趣的内容。此外，您在 Strava 网站的会话期间输入您的会员 ID 和密码后，Strava 会保存该信息，这样您就不必在该会话期间重复输入该信息。大部分 Web 浏览器会自动接受 cookie，但您通常可以更改浏览器来阻止这一操作。不接受 cookie 可能会让您无法使用网站的某些功能。Strava 可能会使用您的 IP 地址来识别您、管理网站及协助诊断 Strava 服务器的问题。

与大多数标准 Web 站点服务器一样，网站会使用日志文件。这其中包含互联网协议 (IP) 地址、浏览器类型、互联网服务提供商 (ISP)、引用/退出页面、平台类型、日期/时间戳记和点击次数，用于分析趋势、管理网站、跟踪用户的总体活动情况，以及收集总体使用情况的广泛人口统计信息。IP 地址等信息不会与可识别个人身份的信息关联起来。此外，Strava 还使用 Flurry、Adjust、HockeyApp 和 Crashlytics 方法来跟踪移动统计信息和应用程序中的用户行为。Android 上的蓝牙 LE 传感器支持是由 Wahoo 管理的，其服务可能会收集一般的设备和连接度量值。

您可以决定在可能包含 Strava 产品或服务相关信息或查询的电子邮件消息中向 Strava 发送可识别个人身份的信息。Strava 将使用该信息来识别您的 Strava 会员身份，以及回复电子邮件。Strava 将仅使用所获取的信息来解决电子邮件中确定的问题。Strava 绝不会出售电子邮件地址。您可以联系 support@strava.com 来更改或请求删除您的个人信息。
